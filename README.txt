== Configurations & Use ==

= Follow the following steps for a fully functional SEO Repair Kit: =

1. After installation, go to "SEO Repair Kit" menu in the admin panel, select post type from dropdown to scan.
2. Go to Settings menu of "SEO Repair Kit" and select/unselect post types and save them for dashboard dropdown.
3. Go to Alt Image Missing menu of "SEO Repair Kit" and add alt text of Media images.
4. Go to Redirection menu of "SEO Repair Kit" and add complete old and new url and save redirection.

== Installation ==

1. Download plugin.
2. Upload `seo-repair-kit.zip` to the `/wp-content/plugins/` directory to your web server.
3. Activate the plugin through the 'Plugins' menu in WordPress.
